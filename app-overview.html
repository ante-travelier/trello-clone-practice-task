Full-Stack Trello Clone

A portfolio-grade project management app built with React, Node.js, and PostgreSQL — complete with auth, drag-and-drop, and a comprehensive test suite.

Features

Everything you need to manage work visually — boards, lists, cards, and more.

🔒

Authentication

Register, login, and auto-refresh sessions. Access tokens live in memory, refresh tokens in httpOnly cookies — secure by default.

📋

Boards

Create, rename, recolor, and delete boards. Each board is a Kanban workspace with its own lists and cards. Six preset board colors.

☰

Lists & Cards

Add lists to boards, add cards to lists. Rename, reorder, move between lists, or delete — with full cascading cleanup.

🎨

Drag & Drop

Reorder lists horizontally and cards vertically (or across lists) with smooth drag-and-drop. Optimistic UI with server rollback.

🏷

Labels

Color-code cards with labels (green, yellow, orange, red, purple, blue). Labels display on both the card tile and the detail modal.

☑

Checklists

Create checklists on any card, add items, and toggle completion. A progress bar shows percentage complete at a glance.

📅

Due Dates

Set due dates on cards. Color-coded badges: red for overdue, yellow for today, gray for future — visible on the board and in the modal.

🚀

Demo Data

New users get two fully seeded boards with realistic cards, labels, and checklists — ready to explore out of the box.

Key Design Decisions

Token Strategy

Access tokens (15 min) stored in React state — never touches localStorage. Refresh tokens (7 days) in httpOnly cookies, immune to XSS.

Float Positions

Lists and cards use float-based positions so reordering only updates one row instead of re-indexing the entire list.

Optimistic Updates

Drag-and-drop applies changes instantly in the UI and rolls back on server error — keeping the experience snappy.
